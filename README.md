## Overview

Tab Choir Audition is a web-based HTML project built to present activities that help with audition preparation for the Tabernacle Choir. Because everything is delivered through the browser, the material is easy to open, review, and work through without needing a complicated setup.

It is a practical starting point for singers and organizers who want audition prep content gathered in one place. The repository keeps the focus on the preparation work itself, reducing overhead so you can spend more time practicing and less time arranging files.
